The code contained in this repo scrapes account data, as well as post data (including all top level comments), from TikTok. Important: A list of already working proxies is requried to use this code.

Notes to the the individual files:

1. video_links_from_pagesource.py: The account's post urls are extracted from the page source of the account. The source code must be downloaded manually. I did not manage to automate this process reliably. 
2. check_proxies.py: Checks if your proxies have a valid connection to TikTok (checks if status code is 200).
3. scraper.py: The scraper needs 3 things to work. The account name (without the @), a list of working proxies and json file with post urls. The json file containing post urls is generated by video_links_from_pagesource.
